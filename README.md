# 👋 Hi, I'm Christian Gallon

🔐 Cloud Security & IAM Engineer focused on building secure, scalable cloud environments across AWS and Azure.

---

## 🧠 About Me

I specialize in identity governance, least privilege access, and cloud security architecture.
My work focuses on solving real-world security problems such as over-permissioned access, lack of visibility, and compliance gaps in cloud environments.

I build projects that simulate enterprise security challenges and implement practical, production-style solutions using Infrastructure as Code and automation.

---

## 🚀 What I Do

* 🔐 Design secure cloud architectures (AWS & Azure)
* 🔑 Implement IAM governance and least privilege access models
* ⚙️ Automate infrastructure and security controls using Terraform
* 📊 Build logging, monitoring, and compliance solutions
* 🛡️ Detect and remediate cloud misconfigurations
